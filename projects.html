Finance Wordle

A Wordle clone I built in collaboration with AI for my IS 201 final project. Instead of common 5-letter English words, this version uses finance vocabulary — things like YIELD, BONDS, ASSET, and TRADE. After each round, the game shows the definition of the word so you can actually learn something. Built as a single HTML file with inline CSS and JavaScript.
