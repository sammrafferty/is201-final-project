# IS 201 Final Project - Sam Rafferty

Personal website built for the IS 201 final web development project at BYU.

## Pages

- `index.html` - Bootstrap Resume theme: my HTML resume (about, experience, education, skills, interests, awards)
- `projects.html` - A second Bootstrap page showcasing my projects
- `scratch.html` - A from-scratch HTML/CSS page about cycling, with an embedded YouTube video, a live Tableau dashboard, and a custom stylesheet
- `app.html` - Finance Wordle: a Wordle clone built in collaboration with AI, with a finance-themed word list
